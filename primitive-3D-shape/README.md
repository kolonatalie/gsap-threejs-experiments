# Three.js PBR & GSAP Interaction Lab | [Live Demo]()

A focused experiment on physically-based rendering (PBR) and smooth animations using Three.js and GSAP.

## Features
- **Interactive Raycasting**: Real-time collision detection that triggers dynamic HSL color shifts on user click.
- **PBR Materials**: Implementing `MeshStandardMaterial` with high metalness and custom roughness.
- **Environment Lighting**: Using `PMREMGenerator` and `RoomEnvironment` for realistic reflections.
- **GSAP Integration**: Entry animations and continuous rotation loops.
- **Interactive Controls**: Smooth navigation via `OrbitControls` with damping.

## Tech Stack
- **Three.js**
- **GSAP**
- **Import Maps**

## Getting Started (Local Setup)

1.  **Clone the repository:**
    
    `git clone` [`https://github.com/kolonatalie/gsap-threejs-experiments.git`](https://github.com/kolonatalie/gsap-threejs-experiments.git)
    
2.  **Navigate to the project folder:**
    ```bash
    cd gsap-threejs-experiments/primitive-3D-shape
    ```
3.  **Run the code:**
    Open the `index.html` file in your browser (Live Server recommended).
